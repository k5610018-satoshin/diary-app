# My Diary - 小学生向けAI日記アプリ

## 📖 概要
小学生が日記を書き、AIからフィードバックを受けられる教育向けアプリケーションです。

## 🚀 起動方法

### 方法1: バッチファイルで起動（簡単）
1. `MyDiary起動.bat` をダブルクリック
2. 自動的にブラウザが開き、アプリが表示されます
3. 終了するにはコマンドプロンプトウィンドウを閉じてください

### 方法2: コマンドラインで起動
```bash
# 依存関係のインストール（初回のみ）
npm install

# 本番ビルド
npm run build

# サーバー起動
npm run start
```
ブラウザで `http://localhost:3000` にアクセス

### 方法3: 開発モード
```bash
npm run dev
```
または `開発モード起動.bat` をダブルクリック

## 📱 機能

### 児童向け機能
- 🖊️ 日記の作成（テキスト・音声入力対応）
- 🤖 AIからのフィードバックと質問
- 📝 追記機能（音声入力対応）
- 🖼️ 画像添付

### 先生向け機能（/admin）
- 👀 全児童の日記一覧表示
- 🔍 日付別・児童別フィルター
- 💬 コメント機能
- 📊 CSVエクスポート（スプレッドシート連携）
- 📄 PDFエクスポート

## ⚙️ 設定

### Gemini API キー設定
1. `/admin` にアクセスしてログイン（パスワード: `teacher2024` または `admin`）
2. 歯車アイコンをクリック
3. Gemini APIキーを入力
4. [APIキーを取得](https://aistudio.google.com/app/apikey)

## 🌐 ネットワーク経由でアクセス

同じネットワーク内の他のデバイスからアクセスする場合：

1. サーバーを起動
2. コマンドプロンプトでIPアドレスを確認: `ipconfig`
3. 他のデバイスのブラウザで `http://[IPアドレス]:3000` にアクセス

例: `http://192.168.1.100:3000`
