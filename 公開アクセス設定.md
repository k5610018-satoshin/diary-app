# 🌐 公開アクセス設定ガイド

インターネット経由でアプリにアクセスするための方法を説明します。

## 方法1: Vercelでホスティング（推奨・無料）

Vercelは Next.js を作ったVercel社の無料ホスティングサービスです。

### 手順

1. **GitHubアカウントを作成**
   - https://github.com にアクセス
   - アカウントを作成

2. **GitHubにコードをアップロード**
   ```bash
   # diary-appフォルダで実行
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/[ユーザー名]/diary-app.git
   git push -u origin main
   ```

3. **Vercelにデプロイ**
   - https://vercel.com にアクセス
   - 「Continue with GitHub」でログイン
   - 「New Project」をクリック
   - GitHubリポジトリをインポート
   - 「Deploy」をクリック

4. **完了！**
   - `https://[プロジェクト名].vercel.app` のようなURLでアクセス可能
   - このURLを共有すれば誰でもアクセスできます

---

## 方法2: ngrok（一時的な公開・無料）

PCを起動している間だけ公開したい場合に便利です。

### 手順

1. **ngrokをインストール**（すでにインストール済み）
   ```bash
   npm install -g ngrok
   ```

2. **ngrokアカウントを作成**
   - https://ngrok.com にアクセス
   - 無料アカウントを作成
   - 認証トークンを取得

3. **認証を設定**
   ```bash
   ngrok config add-authtoken [あなたのトークン]
   ```

4. **サーバーを起動**
   ```bash
   npm run dev
   # または
   npm run start
   ```

5. **別のターミナルでngrokを起動**
   ```bash
   ngrok http 3000
   ```

6. **公開URLを取得**
   - 表示される `https://xxxx.ngrok-free.app` のようなURLを共有
   - このURLでどこからでもアクセス可能

### 注意点
- PCの電源を切ると公開が終了します
- 無料版は8時間の制限があります
- URLは毎回変わります

## 方法3: Cloudflare Tunnel（無料・常時公開）

自宅PCを常時公開サーバーにしたい場合。

詳細: https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/

## 推奨

- **学校での利用**: Vercel（無料、安定）
- **一時的なデモ**: ngrok（手軽）
- **常時運用**: Cloudflare Tunnel または VPS
